## **Superstore Order Management and Sales Analysis**

## 📌 Project Overview
This Power BI project analyzes Superstore sales data to evaluate order management efficiency, sales performance, and profitability trends across regions and product categories.

## 🎯 Business Problem
The company wants to:
- Improve order fulfillment performance
- Identify high-performing regions and products
- Optimize sales and profit margins
- Detect loss-making segments

## 🛠 Tools & Technologies
- Microsoft Power BI
- DAX
- Power Query
- Excel (Data Source)

## 📊 Key Metrics Analyzed
- Total Sales
- Total Profit
- Profit Margin
- Number of Orders
- Average Order Value
- Sales by Region
- Sales by Category & Sub-Category

## 📈 Key Insights
- West region generated the highest revenue.
- Technology category produced the highest profit margin.
- Certain sub-categories showed negative profitability.
- Discount levels significantly impacted profit.
